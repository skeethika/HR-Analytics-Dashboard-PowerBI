#HR Analytics Dashboard using Power BI

##Objective:
To develop an interactive HR Analytics Dashboard that visualizes key workforce metrics such as employee demographics, attrition, hiring, salary, performance, and job satisfaction, enabling data-driven HR decision-making and improving workforce management.
